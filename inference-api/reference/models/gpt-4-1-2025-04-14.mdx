**Model ID:** `gpt-4-1-2025-04-14`

GPT 4.1 is an LLM that excels in coding, instruction following, and handling extremely long contexts—supporting up to one million tokens per interaction. It offers significant improvements over previous models in generating accurate code, following complex instructions, and understanding nuanced prompts within large documents or multi-step workflows.

Some other noteworthy features of GPT 4.1 include enhanced performance for agentic workflows and improved collaboration on tasks requiring detailed reasoning or multi-part instructions.

| Mixture of Experts | No                 |

| Multilingual       | Yes                |

    ```bash cURL
    curl -X POST https://hub.oxen.ai/api/ai/chat/completions \
      -H "Content-Type: application/json" \

      -d '{
      "model": "gpt-4-1-2025-04-14",

      ]
    }'
    ```

    ```python Python
    import os
    import requests
    ```
