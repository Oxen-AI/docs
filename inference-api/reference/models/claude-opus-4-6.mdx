**Model ID:** `claude-opus-4-6`

**Claude Opus 4.6 is a Multimodal LLM designed for coding, enterprise agents, and professional work.** It excels in reasoning, planning, software engineering, agentic tasks, and long context processing, with features like adaptive thinking that adjusts reasoning effort by task complexity.

Some other noteworthy features of Claude Opus 4.6 include image processing for analyzing charts and diagrams, context compaction for extended conversations, and computer use for multi-step workflows.
