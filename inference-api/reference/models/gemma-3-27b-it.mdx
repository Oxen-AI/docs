**Model ID:** `gemma-3-27b-it`

Gemma 3 27B Instruct is a 27 billion parameter multimodal LLM that excels in a wide variety of tasks, including question answering, reasoning, code generation, and image understanding. It features a large 128K token context window and supports over 140 languages, enabling complex analysis and multilingual capabilities.

Some other noteworthy features of Gemma 3 27B Instruct include function calling for task automation, structured output generation, and the ability to process both text and images.

| Metric             | Value              |
|--------------------|--------------------|
| Parameter Count    | 27 billion         |
| Mixture of Experts | No                 |
| Context Length     | 128,000 tokens     |
| Multilingual       | Yes                |
| Quantized*         | No                 |

*_Quantization is specific to the inference provider and the model may be offered with different quantization levels by other providers._
