**Model ID:** `qwen3-vl-8b-instruct`

Qwen/Qwen3-VL-8B-Instruct is a versatile multimodal large language model capable of understanding and generating both text and images. Built on the Qwen3 architecture, it provides strong general reasoning, detailed image interpretation, and instruction-following performance in a compact 8B parameter size.
