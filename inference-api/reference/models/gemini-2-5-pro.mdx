**Model ID:** `gemini-2-5-pro`

Gemini 2.5 Pro Preview is a Multimodal LLM designed for enhanced coding capabilities. It excels in building interactive web applications, front-end and UI development, code transformation, code editing, and developing complex agentic workflows.

Some other noteworthy features of Gemini 2.5 Pro Preview include video-to-code capabilities and native multimodality with strong performance on video understanding, scoring 84.8% on the VideoMME benchmark.

    ```bash cURL
    curl -X POST https://hub.oxen.ai/api/ai/chat/completions \
      -H "Content-Type: application/json" \

      -d '{
      "model": "gemini-2-5-pro",

    }'
    ```

    ```python Python
    import os
    import requests

    response = requests.post(

                {

                    "content": "Hello, what can you do?"

            ]
    ```
