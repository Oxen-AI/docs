**Model ID:** `gemini-3-flash-preview`

**gemini-3-flash-preview is a Multimodal LLM.** It excels in agentic workflows, multi-turn chat, coding assistance, and interactive tasks due to its lower latency and near-Pro reasoning compared to larger Gemini variants.

Some other noteworthy features of gemini-3-flash-preview include configurable thinking levels (minimal, low, medium, high), structured output, tool use, automatic context caching, and support for multimodal inputs like text, images, audio, video, and PDFs.

|--------------------|--------------------|
| Parameter Count    | Unknown            |
| Mixture of Experts | Unknown            |
| Context Length     | 1M tokens          |
| Multilingual       | Yes                |
