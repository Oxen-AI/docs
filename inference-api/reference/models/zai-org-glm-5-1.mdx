**Model ID:** `zai-org-glm-5-1`

zai-org/GLM-5.1 is a Mixture-of-Experts flagship language model from Z AI built for agentic engineering, with strong coding capabilities and sustained performance over long-horizon tasks spanning hundreds of iteration rounds.

GLM-5.1 supports function calling and a long context window, making it well suited for code assistance, agentic systems, and other workflows that require reasoning across many steps. It is offered in an FP8 variant for efficient serving.

| Metric             | Value              |
|--------------------|--------------------|
| Parameter Count    | 743 billion        |
| Mixture of Experts | Yes                |
| Context Length     | 202,000 tokens     |
| Multilingual       | Yes                |
| Quantized*         | Yes                |
| Precision*         | FP8                |

*_Quantization is specific to the inference provider and the model may be offered with different quantization levels by other providers._
