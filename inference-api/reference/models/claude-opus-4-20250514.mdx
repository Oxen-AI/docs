**Model ID:** `claude-opus-4-20250514`

Claude Opus 4 is a Multimodal LLM designed for reasoning-heavy tasks and complex coding workflows. It excels in sustained performance on long-running tasks that require focused effort and thousands of steps, including agentic search, large-scale code refactoring, and multi-step problem solving.

Some other noteworthy features of Claude Opus 4 include "extended thinking" mode for more deliberate reasoning, tool use capabilities, and memory tracking across complex workflows.
