title: "Domains"
---

Here lives all the HTTP APIs for all the Domains in [oxen-server](/getting-started/oxen-server).

- [Namespaces](/http-api/namespaces/list_namespaces): Namespaces are a way to organize groups of repositories.
- [Repositories](/http-api/repositories/list_repos): Repositories are the primary way to store and manage data in Oxen.
- [Branches](/http-api/branches/get_branches): Branches are a way to manage different versions of a repository.
- [Commits](/http-api/commits/list_commits): Commits are a way to manage different versions of a repository.
- [Files](/http-api/files/list_entries): List the files and folders in a repository, given a specific revision.
- [Data Frames](/http-api/data_frames/get_data_frame): Data Frames are a way to manage tabular data in a repository.
