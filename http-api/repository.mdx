---
title: "Get Repositories"
api: "GET https://hub.oxen.ai/api/repos/:namespace"
---

Get repositories information under a namespace.

```bash
curl -X GET \
    https://hub.oxen.ai/api/repos/:namespace
```

--- 
title: "Create Repository"
api: "POST https://hub.oxen.ai/api/repos"
---


Parameters:
- name: repo name
- description: describe the repo
- is_public(optional): private repo by default 
- files(optional): create repository with files 
- organization(optional): create repository in an organization

## Example Request

```bash
curl -X POST \
    https://hub.oxen.ai/api/repos \
    --data '{
    "name": "repo",
    "description": "Aut dolore voluptatibus dolore adipisci iusto iste non quis. Voluptas ut eos sit error. Illo tempore error ipsam occaecati eos quia incidunt. Dicta labore quaerat amet autem sequi voluptas amet voluptate aut. Voluptatem et sunt vel quo amet quos cupiditate modi quasi.",
    "is_public": true,
    "organization_id": "{{org_id}}",
    "files": [
        {
            "path": "README.md",
            "contents": "# My Repo Name",
            
            "user": {
                "name": "ox",
                "email": "ox@oxen.ai"
            }
        }
    ]
}'
```

---
title: "Delete Repository"
api: "DELETE https://hub.oxen.ai/api/repos/:namespace/:repo_name"
---

Delete a repo.

```bash
curl -X DELETE \
    https://hub.oxen.ai/api/repos/:namespace
```

---
title: "Update Repositories"
api: "PUT https://hub.oxen.ai/api/repos/:namespace"
---

Update repository information.

```bash
curl -X PUT \
    https://hub.oxen.ai/api/repos/:namespace \
    -- data '{
    "description": "Add some description",
    "is_public": false,
    "default_branch": "main"
    }'
```
