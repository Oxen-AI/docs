---
title: "Get Repositories"
api: "GET https://hub.oxen.ai/api/repos/:namespace"
---

You can get public repositories information under a namespace without authentication.

```bash
curl -X GET \
    https://hub.oxen.ai/api/repos/:namespace
```

Or get info about a single repo.

```bash
curl -X GET \
    https://hub.oxen.ai/api/repos/:namespace/:repo_name
```
