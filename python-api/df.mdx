# oxen.df

The `df` module provides a consistent interface for working with data frames.

Supported types: csv, parquet, json, jsonl, arrow

Example usage:

```python
import os
from oxen import df

# load a data frame
data_frame = df.load("path/to/data.csv")

# save a data frame
df.save(data_frame, "path/to/save.csv")
```
